# Token申请地址：
[https://alltick.co/register](https://alltick.co/register)

# 申请流程
## 第1步：在Token申请地址页面依次填写邮箱，密码（注意大小写，特殊字符，数字），邮箱验证码
## 第2步：填写完毕，点击"注册"按钮，等待片刻就会申请完毕
## 第3步：申请完毕，页面会回到仪表盘，就会进入dashboard界面，在"API keys"就能找到你的Token了
